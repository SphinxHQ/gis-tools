## 二、采集方法（修订）

### 2.1 核心方法

| 项目 | 内容 |
|---|---|
| 方法 | evaluate_script 组件白名单遍历 |
| 输出 | JSON 文件 |
| 字段 | component, tag, rect(w/h/x/y), scrollable, overflowY, margin, padding, color, bg |

### 2.2 JSON 数据文件命名

```
scene{N}-{viewport}.json
例：
scene1-500x667.json   (场景1, xs视口)
scene1-600x960.json   (场景1, sm视口)
scene1-768x1024.json  (场景1, md视口)
scene1-1024x768.json  (场景1, lg视口)
scene1-1280x800.json  (场景1, xl视口)
scene1-1920x1080.json (场景1, xx视口)
```
