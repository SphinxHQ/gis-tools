## 四、关键发现（DOM 元素特征）

### 4.1 top-bar 高度随视口变化
- xs/sm (移动): 48px
- md: 44px
- lg/xl/xx (桌面): 40px

### 4.2 main-content 占比
- 桌面端 (lg+): top-bar 40px + main 剩余高度
- 移动端 (sm): top-bar 48px + gis-mobile-nav 56px + status-bar 24px

### 4.3 left-panel（数据加载后才出现）
- xl/xx: 521px
- lg: 521px
- md: 321px（窄屏）
- sm/xs: 不可见（移动端导航）

### 4.4 map-container 与 left-panel 互斥
- xl/xx: map 1395px
- lg: map 499px（left 521px + map 499px = 1020px ≈ 1024）
- md: map 447px
- sm/xs: map 全宽（500/600）

### 4.5 滚动元素
- `gis-data-transformer-panel`: 所有桌面视口都设置 `overflow-y: auto`
- 其他主要容器多 `overflow-y: hidden`（裁切溢出）

### 4.6 数据加载前后的元素数量对比
- 场景1-6 (无数据): 15-20 元素/视口
- 场景7-8 (有数据): 47-95 元素/视口（增加 left-panel、el-segmented、crs-info-card 等）
