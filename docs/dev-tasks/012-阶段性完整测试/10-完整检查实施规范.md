## 一、采集方法（重要修订）

### 1.1 核心方法：evaluate_script 组件白名单遍历

**脚本注入**：
```javascript
// 先注入组件名称识别函数
window.__getComponentName = function(el) {
  if(!el) return null;
  var name = el.getAttribute('data-component');
  if(name) return name;
  if(el.className && typeof el.className === 'string') {
    var classes = el.className.split(' ');
    for(var i=0; i<classes.length; i++) {
      var c = classes[i];
      if(c.match(/^(el-|gis-|top-|left-|main-|empty-|upload-|map-|mobile-|dialog-|crs-|feature-|validator-|export-|data-|history-)/)) {
        return c;
      }
    }
  }
  return el.tagName.toLowerCase();
};
```

**采集脚本**：
```javascript
var vp = window.innerWidth + 'x' + window.innerHeight;
var r = function(el) {
  if(!el) return null;
  var rect = el.getBoundingClientRect();
  var s = window.getComputedStyle(el);
  if(rect.width < 1 || rect.height < 1) return null;
  var cn = 'unknown';
  if(el.className && typeof el.className === 'string') {
    var cls = el.className.split(' ');
    for(var i = 0; i < cls.length; i++) {
      var c = cls[i];
      if(c.match(/^(el-|gis-|top-|left-|main-|empty-|upload-|map-|mobile-|dialog-|crs-|feature-|validator-|export-|data-|history-)/)) {
        cn = c;
        break;
      }
    }
  }
  return {
    component: cn,  // 组件名称
    tag: el.tagName,
    rect: {
      w: Math.round(rect.width),
      h: Math.round(rect.height),
      x: Math.round(rect.left),
      y: Math.round(rect.top)
    },
    scrollable: el.scrollHeight > el.clientHeight,
    overflowY: s.overflowY,
    margin: s.marginTop + ' ' + s.marginRight + ' ' + s.marginBottom + ' ' + s.marginLeft,
    padding: s.paddingTop + ' ' + s.paddingRight + ' ' + s.paddingBottom + ' ' + s.paddingLeft,
    color: s.color,
    bg: s.backgroundColor
  };
};

// 组件白名单（按优先级）
var tags = ['div','button','span','p','a','svg','header','nav','main','aside','canvas','[role=dialog]','[role=tab]','[role=tree]','[role=alert]','[role=tooltip]'];

var seen = {};
var result = [];
tags.forEach(function(sel) {
  try {
    var els = document.querySelectorAll(sel);
    els.forEach(function(el) {
      var key = (el.className || '') + '_' + el.tagName + '_' + Math.round(el.getBoundingClientRect().left) + '_' + Math.round(el.getBoundingClientRect().top);
      if(!seen[key] && el.offsetParent !== null && el.offsetParent !== undefined) {
        seen[key] = true;
        var data = r(el);
        if(data) result.push(data);
      }
    });
  } catch(e) {}
});

return JSON.stringify({
  viewport: vp,
  count: result.length,
  elements: result
}, null, 2);
```

## 三、检查项定义

### 3.1 检查项 #1：滚动条情况

**来源字段**：`scrollable`, `overflowY`

**判断逻辑**：
- `scrollable = true` 且 `overflowY != 'hidden'` → 有滚动条
- 否则 → 无滚动条

### 3.2 检查项 #2：DOM 遮挡

**来源字段**：`rect.x`, `rect.y`, `rect.w`, `rect.h`

**判断逻辑**：通过元素坐标计算重叠区域

### 3.3 检查项 #3：边界超出

**来源字段**：`rect.x`, `rect.y`, `rect.w`, `rect.h`

**判断逻辑**：
- `rect.x < 0` → 左超出
- `rect.y < 0` → 上超出
- `rect.x + rect.w > viewport.w` → 右超出
- `rect.y + rect.h > viewport.h` → 下超出

### 3.4 检查项 #4：文字换行

**来源字段**：需额外采集 `scrollHeight` vs `clientHeight`

### 3.5 检查项 #5：文字遮挡

**来源字段**：需额外采集 `z-index` 和重叠检测

### 3.6 检查项 #6：Grid/Flex 对齐

**来源字段**：需额外采集 `display`, `align-items`, `justify-content`

### 3.7 检查项 #7：Margin/Padding

**来源字段**：`margin`, `padding`

### 3.8 检查项 #8：DOM 颜色

**来源字段**：`color`, `bg`

### 3.9 检查项 #9：DOM 实际出现

**来源字段**：`rect.w > 0 && rect.h > 0`

## 五、实施流程

### 5.1 标准流程

```
1. navigate_page(首页)
2. For viewport in [xs, sm, md, lg, xl, xx]:
   a. resize_page(viewport)
   b. evaluate_script(采集脚本)
   c. 保存为 scene{N}-{viewport}.json
3. navigate_page(首页)  // 刷新重置
4. 执行交互操作
5. 重复步骤2
```

### 5.2 交互操作清单

| 场景 | 操作 |
|---|---|
| 场景1 | 页面初始加载（无操作） |
| 场景2 | 点击导入按钮 |
| 场景3 | 停留在上传文件Tab |
| 场景4 | 点击输入文本Tab |
