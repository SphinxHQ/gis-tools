## 一、阻塞中项（依赖真机，来自 911）

### L3-001 ⚠️ Bottom Sheet 拖拽至吸附点的触摸交互

| 字段 | 内容 |
|---|---|
| 编号 | L3-001 |
| 来源 | 911 §1.1 / 第二期 S7 验收 |
| 现象 | MCP/Playwright 无法模拟真实触摸拖拽手势，吸附点交互逻辑未在真机验证 |
| 影响 | 用户真机体验可能存在手势冲突或吸附点不生效 |
| 测试方法 | 真机拖拽 BottomSheet 手柄至各吸附点，验证吸附行为 |
| 预期结果 | 拖拽过程流畅，松手后吸附到最近吸附点，无卡顿/误触 |
| 状态 | ⏳ 待真机 |

### L3-002 ⚠️ Tab 切换状态保留（v-show）真机验证

| 字段 | 内容 |
|---|---|
| 编号 | L3-002 |
| 来源 | 911 §1.2 / 第二期 S7 验收 |
| 现象 | `v-show` 替代 `v-if` 已实施，但状态保留效果（表单输入、滚动位置）未真机验证 |
| 影响 | 可能存在 Tab 切换后状态丢失的边缘情况 |
| 测试方法 | 真机切换 4 个 Tab（坐标系/要素/校验/导出），验证状态保留 |
| 预期结果 | 切换后表单输入、滚动位置、选中状态保留 |
| 状态 | ⏳ 待真机 |

### L3-003 ⚠️ 地图手势与面板手势冲突验证

| 字段 | 内容 |
|---|---|
| 编号 | L3-003 |
| 来源 | 911 §1.3 / 第二期 S7 验收 |
| 现象 | BottomSheet 手柄 `@touchstart` 独立处理，但地图区域与面板边缘的手势冲突未真机验证 |
| 影响 | 用户在 Sheet 边缘操作时可能误触地图缩放/平移 |
| 测试方法 | 真机在 Sheet 展开/收起过程中测试地图手势隔离 |
| 预期结果 | Sheet 边缘操作不误触地图，地图区域手势正常 |
| 状态 | ⏳ 待真机 |
