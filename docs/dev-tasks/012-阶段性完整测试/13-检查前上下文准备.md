## 二、关键上下文要点

### 2.1 三层验证模型（来自 912）

```
L1 静态验证 (必须 100% 自动通过)
    - TypeScript 编译: npx vue-tsc --noEmit
    - 构建: npx vite build
    - VS Code 诊断: GetDiagnostics

L2 运行时验证 (尽量自动化)
    - DOM 状态、布局尺寸、事件响应
    - 工具: mcp_chrome-devtools

L3 真机验证 (标注"待真机")
    - 触摸手势、拖拽手感、性能感知

```

## 三、检查方式方法

### 3.1 检查方法分类

| 检查项 | 检查方法 | 工具 |
|---|---|---|
| #1 滚动条 | scrollHeight > clientHeight + overflow != 'hidden' | evaluate_script |
| #2 DOM 遮挡 | getBoundingClientRect 重叠检测 | evaluate_script |
| #3 边界超出 | 元素超出视口或父容器边界 | evaluate_script |
| #4 文字换行 | clientHeight < scrollHeight | evaluate_script |
| #5 文字遮挡 | z-index + 边界重叠 | evaluate_script |
| #6 Grid/Flex 对齐 | getComputedStyle + 子元素边界 | evaluate_script |
| #7 Margin/Padding | getComputedStyle.margin/padding | evaluate_script |
| #8 DOM 颜色 | getComputedStyle.color/backgroundColor | evaluate_script |
| #9 DOM 实际出现 | getBoundingClientRect 宽高 > 0 | evaluate_script |

### 3.2 检查流程（每个元素 × 每个视口）

```
1. resize_page(target_viewport)
2. evaluate_script(element_data_collector)
3. 返回原始数据 (JSON)
4. 填入对应 .md 模板
```

### 3.3 9 项检查的数据采集脚本框架

```javascript
function collectElementData(selector, viewport) {
  const el = document.querySelector(selector);
  if (!el) return { found: false };
  
  const rect = el.getBoundingClientRect();
  const style = window.getComputedStyle(el);
  
  return {
    viewport,
    selector,
    rect: {
      x: rect.x, y: rect.y, 
      width: rect.width, height: rect.height
    },
    
    // #1 滚动条
    scrollable: el.scrollHeight > el.clientHeight,
    scrollHeight: el.scrollHeight,
    clientHeight: el.clientHeight,
    overflowY: style.overflowY,
    
    // #3 边界超出
    beyondViewport: {
      top: rect.top < 0,
      bottom: rect.bottom > window.innerHeight,
      left: rect.left < 0,
      right: rect.right > window.innerWidth
    },
    
    // #4 文字换行
    textWrap: el.scrollHeight > el.clientHeight + 2,
    
    // #7 Margin/Padding
    margin: {
      top: style.marginTop,
      right: style.marginRight,
      bottom: style.marginBottom,
      left: style.marginLeft
    },
    padding: {
      top: style.paddingTop,
      right: style.paddingRight,
      bottom: style.paddingBottom,
      left: style.paddingLeft
    },
    
    // #8 颜色
    color: style.color,
    backgroundColor: style.backgroundColor,
    borderColor: style.borderColor,
    
    // #9 实际出现
    found: rect.width > 0 && rect.height > 0
  };
}
```
