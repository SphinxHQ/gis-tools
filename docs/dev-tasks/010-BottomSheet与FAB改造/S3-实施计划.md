### M1: 新建 useDataPanelTabs composable
- **文件**: `src/composables/useDataPanelTabs.ts`（新建）
- **修改内容**: 提取 GisDataPanel 与 GisMobileNav 共享的 Tab 状态管理逻辑

### M2: 新建 GisBottomSheet 组件
- **文件**: `src/components/data/GisBottomSheet.vue`（新建）
- **修改内容**: 通用底部 Sheet 容器组件
  - Props: `visible`(boolean)、`snapPoints`(number[], 默认 [25, 40, 80] vh 百分比)、`defaultSnap`(number, 默认 40)
  - Events: `update:visible`、`height-change`(heightPx: number)、`state-change`(state: 'collapsed'|'half'|'expanded')
  - 功能：
    - 拖拽手柄（视觉 36×4px，热区全宽×32px，`touchstart/touchmove/touchend` + `mousedown/mousemove/mouseup`）
    - 吸附点动画（300ms Standard easing `cubic-bezier(0.4, 0.0, 0.2, 1)`，使用 `transform: translateY()`）
    - 遮罩层（点击关闭，半透明背景）
    - 安全区域适配（`padding-bottom: env(safe-area-inset-bottom, 0px)`）
    - `prefers-reduced-motion` 支持（动画压缩到 0-50ms）
    - `requestAnimationFrame` 驱动拖拽（60fps）
- **影响面**: 新文件，无现有依赖。被 M4 使用
- **验证方法**: 
  - L3-08 手柄热区 32px、可拖拽
  - L3-09 吸附点 [25vh, 40vh, 80vh]、300ms 动画
  - L5-17 拖拽帧率 ≥55fps、使用 transform
- **回滚方式**: 删除文件

## 风险与回滚总述

### 风险
1. **触控事件兼容性**：Bottom Sheet 拖拽需同时支持 touch 和 mouse 事件，iPad/Android/iOS 行为可能差异
   - 缓解：S6 阶段 MCP Playwright 触控模拟 + 真机抽测
2. **v-show 内存占用**：4 个 Tab 组件同时挂载，内存占用增加
   - 缓解：4 个组件均为轻量级，内存增量可接受（<5MB）
3. **FAB 遮挡地图**：FAB 位于右下角可能遮挡地图要素
   - 缓解：FAB 56dp + 16dp 边距，遮挡面积 <3% 视口；用户可拖拽地图避开

### 回滚策略
- 单项回滚：每个修改项独立，可单独 git checkout 恢复
- 整体回滚：`git checkout feature/phase2-bottom-sheet-fab~1 -- src/` 恢复全部二期改动
- 文档保留：回滚不影响 docs/dev-tasks/010 文档，便于后续恢复
