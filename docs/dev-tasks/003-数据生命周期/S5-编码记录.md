## 各修改项实施详情

### M1: Store 结构重构

**文件**: `src/composables/gisDataStore.ts`

改动内容：
- `GisDataSetEntry` 新增 `sourceId: string` 和 `appendedFrom?: { name: string; count: number }[]`
- 新增 `GisDataSource` 接口 `{ id, name, datasets: GisDataSetEntry[] }`
- 新增 `activeSourceId` ref
- 新增 `addDataSource(dataInfo)` — 创建新数据源 + 首个数据集
- 新增 `updateDataset(id, data)` — 更新现有数据集内容
- 修改 `addDataset(dataInfo, sourceId?)` — 可选指定源，默认归到 activeSourceId
- 修改 `appendToDataset` — 追加后写入 `appendedFrom` 记录
- 修改 `removeDataset` — 删除空数据源时更新 activeSourceId
- 修改 `setActive` — 同步激活数据源
- 修改 `clearAll` — 同时清空 activeSourceId
- 新增 `dataSources` computed — 从扁平 datasets 派生数据源分组

### M5: 父组件适配

**文件**: `src/components/data/GisData.vue`

改动内容：
- 解构 `addDataSource` 替代 `addDataset`
- `handleRead()`: `addDataset(dataInfo)` → `addDataSource(dataInfo)`
- `handleDrop()`: `addDataset(data)` → `addDataSource(data)`

### M2: 导入弹窗改造

**文件**: `src/components/data/GisDataImportDialog.vue`

改动内容：
- 解构 `addDataSource, appendToDataset, datasets, dataSources`
- 非追加模式：`addDataset(data)` → `addDataSource(data)`
- 追加模式：新增 `appendTargetId` ref + `el-select` 目标选择器（按数据源分组）
- 追加模式 `handleRead`：`appendToDataset(appendTargetId, data)` 替代 `appendToDataset(activeId, data)`

### M3: 数据源树形面板适配

**文件**: `src/components/data/GisDataPanel.vue`

改动内容：
- 解构 `dataSources, activeSourceId`
- `sourceTreeData` computed 改为从 `dataSources` 派生真实两级树（多数据源根节点）
- `SourceTreeNode` 接口新增 `appendedFrom` 字段
- 树节点模板新增"已追加"标记（el-tag warning + el-tooltip 显示追加详情）

### M4: CRS 转换提示更新/另存

**文件**: `src/components/data/GisDataTransformer.vue`

改动内容：
- 解构 `addDataset, updateDataset, activeId, activeSourceId`
- 新增 `promptUpdateOrSaveAs(transformedData)` 函数：
  - ElMessageBox.confirm 提示"更新当前数据" vs "另存为新数据集"
  - 更新 → `updateDataset(activeId, data)`
  - 另存 → `addDataset(data, activeSourceId)`
- `addTransformVersion()` 中 `addDataset(transformedClone)` → `promptUpdateOrSaveAs(transformedClone)`
- `handleResetCrs()` 中 `addDataset(resetClone)` → `promptUpdateOrSaveAs(resetClone)`

### M6: 要素编辑提示更新/另存

**文件**: `src/components/data/GisFeatureTree.vue`

改动内容：
- 引入 `useGisDataStore`，解构 `updateDataset, addDataset, activeId, activeSourceId`
- 新增 `hasUnsavedChanges` ref — 编辑变更跟踪标志
- 新增 `promptUpdateOrSaveAs()` 函数（同 M4 逻辑）
- `handleEditorModifyChange` — 设置 `hasUnsavedChanges = true`
- `handleEditorDataChanged` — 设置 `hasUnsavedChanges = true`
- `handleViewChange` 退出编辑时调用 `promptUpdateOrSaveAs()`
- `handleEditorExit` 调用 `promptUpdateOrSaveAs()`
