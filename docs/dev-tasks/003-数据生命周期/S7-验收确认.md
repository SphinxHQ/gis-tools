# S7: 验收确认 — 数据生命周期架构

## 修改摘要

本次任务实现了数据生命周期架构重构，核心变更：

1. **M1 — Store 结构重构**: `gisDataStore.ts` 从扁平 `datasets[]` 重构为 `dataSources[]`（computed 从扁平数组按 `sourceId` 分组派生），新增 `addDataSource`/`updateDataset`/`appendToDataset` 等 API
2. **M2 — 导入弹窗改造**: `GisDataImportDialog.vue` 非追加模式调用 `addDataSource` 创建新数据源；追加模式新增目标选择器（`el-option-group` 按数据源分组）
3. **M3 — 树面板两级层级**: `GisDataPanel.vue` 从 `dataSources` 派生真实两级树，数据集节点显示"已追加"标记（`el-tag` + `el-tooltip`）
4. **M4 — CRS 转换提示**: `GisDataTransformer.vue` 转换/重设后弹出 `ElMessageBox.confirm`，用户选择"更新当前数据"或"另存为新数据集（同源）"
5. **M5 — 父组件适配**: `GisData.vue` 导入入口（`handleRead`/`handleDrop`）改用 `addDataSource`
6. **M6 — 要素编辑提示**: `GisFeatureTree.vue` 编辑退出时若有未保存变更，弹出同上提示

涉及文件（7 个）：
- `src/composables/gisDataStore.ts`（重写）
- `src/components/data/GisData.vue`（3 处调用改）
- `src/components/data/GisDataImportDialog.vue`（重写）
- `src/components/data/GisDataPanel.vue`（树数据源+标记）
- `src/components/data/GisDataTransformer.vue`（提示函数）
- `src/components/data/GisFeatureTree.vue`（提示函数+变更跟踪）
- `src/components.d.ts`（自动生成）
