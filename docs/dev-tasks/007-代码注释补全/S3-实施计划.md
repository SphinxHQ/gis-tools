## JSDoc 模板

### 文件头部
```typescript
/**
 * @file 文件简述
 * @description 文件详细描述
 * @author yuanyu <yuanyu@supermap.com>
 * @date YYYY-MM-DD
 */
```

### 方法注释
```typescript
/**
 * 方法描述
 * @param paramName - 参数描述
 * @returns 返回值描述
 */
```

### 类型注释
```typescript
/**
 * 类型描述
 */
interface TypeName {
  /** 属性描述 */
  field: string
}
```
